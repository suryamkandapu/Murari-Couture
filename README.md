# Project G

A premium boutique blouse design website built with Next.js.

## Features

- Elegant, modern design with animations
- Responsive layout
- Static data-driven
- WhatsApp integration for inquiries

## Tech Stack

- Next.js 14 (App Router)
- JavaScript (JSX)
- Tailwind CSS
- Framer Motion
- ESLint

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js app router pages
- `components/` - Reusable React components
- `lib/` - Static data and utilities
- `styles/` - Global styles
- `public/` - Static assets

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```
